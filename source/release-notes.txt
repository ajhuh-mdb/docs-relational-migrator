Relational Migrator 1.1.3
-------------------------

*Released July 26, 2023*

Bug Fixes:

- Fixed an issue where data verification after a sync job would sometimes 
  abort, or show as not started.
- Added support for connecting to a SQL Server Named Instance.
- Added collection names to JSON Schema files created by Code Generation feature.
- Fixed an issue where JSON columns could not be migrated if they contained 
  top-level arrays.
- Fixed an issue where excluded fields could overwrite calculated fields 
  with the same name.
- Fixed an issue where calculated fields based on decimal column values 
  would return NaN.
- Fixed an issue where some fields could be cut off in diagram cards after 
  changing key handling strategy.

Relational Migrator 1.1.2

*Released Jun 22, 2023*

Relational Migrator 1.1.2 is now generally available.
