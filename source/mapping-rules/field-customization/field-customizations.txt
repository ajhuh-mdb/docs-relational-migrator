====================
Field Customizations
====================

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol


Field Customization are used to convert your source relational data 
types into customized MongoDB fields.

Use Cases
---------

You can use field customization to perform the following actions:

- Change a source relational datatype to a specific MongoDB BSON datatype.

- Configure :guilabel:`Null Handelings` behavior.

- Rename source table names.
