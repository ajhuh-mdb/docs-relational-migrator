Before you Begin
----------------

Relational Migrator allows you to bring your relational workloads 
to MongoDB with confidence.

To get familiar with Relational Migrator its capabilities, 
features, and use cases review the follow resources:

- `Relation Migrator Learning Byte <hhttps://learn.mongodb.com/courses/mongodb-relational-migrator>`__.
- `Relational Migrator NYC.Local Product Demo <https://www.youtube.com/watch?v=-ND6_m397OM&ab_channel=MongoDB>`__.

Overview
--------

The following steps are the high level workflow of how you can use 
Relational Migrator to get your data into MongoDB. These steps do not 
cover all features or capabilities of Relational Migrator.

.. procedure::

   .. step:: Install

      Install Relational Migrator.

   .. step:: Create a Project

      Create a project, you can use a preloaded sample schema to avoid
      connecting to an existing relational database instance.

   .. step:: Use Recommended Schema

      Use recommended schema, this will automatically create mapping 
      rules based on your relational schema.

   .. step:: Customize your mapping rules

      Add or update mapping rules to match the destination data model
      that best matches your application needs.

   .. step:: Run a Sync Job

      Run a sync job.
