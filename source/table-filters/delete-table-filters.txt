====================
Delete Table Filters
====================

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

You can delete existing table filters.
Deleting table filters remove SQL where and row limit clauses
when pulling data from your table.

Before you Begin
----------------

Your table must have at least one table filter in order to delete one.

Steps
-----

.. procedure::
   :style:  normal

   .. step:: Apply a table filter

      #. From the :guilabel:`Schema model` pane, click the table 
         that has the table filter you want to delete.
      #. On the :guilabel:`Mappings` pane in the :guilabel:`Table Filter` 
         section, click the :icon-lg:`Trash` icon next to the table 
         filter you want to delete.

   .. step:: Confirm deletion of your table filter

      Click the red :guilabel:`Delete` button to confirm deletion of 
      the table filter.

      .. note::

         It can take a few seconds for the table filter to be removed 
         from your project.
