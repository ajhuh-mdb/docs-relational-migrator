====================
Delete Table Filters
====================

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

You can delete existing table filters.
Deleting table filters remove SQL where and row limit clauses
when pulling data from your table.

Before you Begin
----------------

Your table must have at least one table filter in order to delete one.
For details on how to create a table filter, see :ref:`rm-apply-table-filters`.

Steps
-----

.. procedure::
   :style:  normal

   .. step:: Select a table filter for deletion

      a. From the :guilabel:`Schema model` pane, click the table 
         that has the table filter you want to edit.
      #. On the :guilabel:`Mappings` pane in the :guilabel:`Table Filter` 
         section, click the :icon-lg:`Edit` icon next to the table 
         filter you want to edit.

   .. step:: Enter the updated SQL

      a. On the :guilabel:`Edit Filter` pane, under the :guilabel:`SQL Query`
         header, enter the updated SQL where statement for your table filter.
      #. (Optional) You can also add or remove a row limit by clicking
         the :guilabel:`No Limit` radio button or :guilabel:`Limit Number Of Rows`
         and providing a value in the row limit text field.

   .. step:: Save the updated table filter

      Click the green :guilabel:`Save and Close` button to save the 
      updated table filter.
